Luvddit is a clone project inspired by the popular online community platform, Reddit. It provides users with a platform to share, discuss, and vote on a wide range of topics organized into individual communities known as "subreddits". Users can create posts, comment on existing posts, upvote or downvote content, and engage in discussions.

Feature List:

Posts – Full Crud:  
Users should be able to view all posts.  
Users should be able to create a post.  
Users should be able to update their post. 
Users should be able to delete their post.

Communities – Partial Crud:  
Users should be able to view all Communities.  
Users should be able to create a Community.
Users should be able to update their Community. 

Votes – Full Crud:  
Users should be able to view all votes on a post.  
Users should be able to create a vote for a post.  
Users should be able to update their vote for a post. 
Users should be able to delete their vote from a post.

Comments – Full Crud:  
Users should be able to view all comments added to their post.  
Users should be able to add comments to their post. 
Users should be able to update their comment for a post.  
Users should be able to remove comments from their post.

Memberships – Full Crud:  
Users should be able to view all members in their community.  
Users should be able to add a memberships. 
Users should be able to update memberships.
Users should be able to remove memberships.

Search – Bonus  
Users should be able to search for posts/communities/people.  
Users should be able to view the results of their search.

Profie – Bonus  
Users should be able to view their posts/communities. 
Users should be able to view their personal information.
